🔍 How does the list work?

Redis Lists are ordered sequences of strings. Because we use LPUSH (left push), the most recently registered student appears first in the list.

LRANGE students:names 0 -1 returns every element from index 0 (first) to -1 (last). Negative indices count from the end.
