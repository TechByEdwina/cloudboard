⚠️ This Action Is Irreversible

Clicking Reset All Data will:

  • Delete the visitor counter (stats:visitor_count)
  • Delete all student names (students:names)
  • Delete all statistics keys (stats:*)
  • … and every other key in Redis

This is the same as running redis-cli FLUSHALL in the terminal.

🔍 What does FLUSHALL do?

Redis supports multiple databases (numbered 0–15 by default). FLUSHDB wipes only the current database, while FLUSHALL wipes all databases.

In a production system you would never use FLUSHALL — you would instead use DEL or UNLINK with specific key names. This page exists purely to make classroom resets easy.

🔍 Try it in redis-cli

docker compose exec redis redis-cli
127.0.0.1:6379> KEYS *
127.0.0.1:6379> FLUSHALL
127.0.0.1:6379> KEYS *
